Event Board UI Review

Three directions for a shared Luma event board, styled after Luma’s clean editorial UI — white canvas, black CTAs, neutral typography. Sample event: AI Demo Night.

Updated Redesigned for readability: no gradient backgrounds, no floating animations, one accent color. Color comes from event cover art only.
Option A

Clean Feed

Full-width event cards, minimal header, black pill buttons. Closest to Luma’s event discovery list.

 Option B

Social Row

Compact cards with a “who’s going” row up front. Social proof without rainbow avatars or warm tints.

 Option C

Event Page

Hero cover + detail layout inspired by a Luma event page. Secondary events as simple list rows.

Best at ~390px width. Serve via python3 -m http.server 8080 in this folder.
