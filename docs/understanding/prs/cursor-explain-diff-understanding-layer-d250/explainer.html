At a glance

This PR adds a human-in-the-loop understanding layer: literate diffs, quizzes, optional micro-worlds, and a numbered reading order so you can review agent code incrementally.

Background

Deep background (skippable)

Long agent runs produce many commits. Raw diffs are hard to retain. Andy Matuschak: reading alone often fools us into thinking we understood. This PR adds teaching docs alongside code — generated deliberately via the understanding skill bundle, not a git hook.

What mattered before

The repo had incremental commits and InsForge skills, but no process for humans to learn and verify agent work as it lands.

Intuition

Goal: Run one understanding task → classify commits → write explainers for substantive ones → maintain reading-order.md → PR roll-up.
