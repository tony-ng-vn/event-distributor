At a glance

Adds HTML/Markdown templates with interactive quiz JS, plus understanding-diff.mjs to gather git context for agents.

Background

Deep background (skippable)

Templates give consistent structure; the script outputs diff-context.json so agents write facts before prose.

What mattered before

Commits #1–#2 had instructions but no scaffolds or npm command.

Intuition

Goal: One command → diff context → agent fills template.

npm run understanding:diff→diff-context.json→explainer.html

HTML template includes working quiz JavaScript. Code blocks use pre-wrap so newlines don't collapse.
