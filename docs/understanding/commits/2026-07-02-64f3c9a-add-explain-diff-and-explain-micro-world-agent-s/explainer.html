At a glance

Adds the first building blocks: explain-diff and explain-micro-world agent skills — playbooks that tell Cursor agents how to write teaching documents about code.

Background

Deep background (skippable)

A git diff shows line changes but rarely explains why or how pieces connect. An agent skill is a markdown file in .cursor/skills/ that agents read when relevant — like a playbook.

What mattered before

The repo had incremental commit rules but no process for teaching humans what agent code does.

Intuition

Goal: Give the agent recipes for literate diffs (essays + quiz) and micro-worlds (playgrounds you choose before the agent builds).

explain-diff+explain-micro-world→teaching layer

This commit is instructions only — templates and scripts land in commit #3.
